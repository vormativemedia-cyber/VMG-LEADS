Woher bekomme ich die Firmendaten?

Dieses System durchsucht nicht automatisch das Internet — es organisiert, filtert und priorisiert Leads, die du reinziehst. Gute Quellen für Rohdaten in Deutschland:

  • Google Maps / Places — lokale Betriebe filtern nach "keine Website" (kein Website-Link im Eintrag)
  • Handelsregister / Unternehmensregister / North Data — Neugründungen & Geschäftsführer-Namen
  • Gelbe Seiten / Das Örtliche — Branchenlisten mit Telefonnummern
  • LinkedIn Sales Navigator — Startups, Gründer, Ansprechpartner
  • IHK-Verzeichnisse — regionale Unternehmenslisten nach Branche

Auto-Suche

Lokale Betriebe automatisch über Google Places finden

Wie das funktioniert

Diese Funktion nutzt die offizielle Google Places API mit deinem eigenen API-Key. Sie durchsucht Google Maps nach Betrieben in deiner Zielregion, prüft je Treffer ob eine Website hinterlegt ist, und legt Treffer ohne Website automatisch als neuen Lead an. Name, Adresse und Telefonnummer kommen aus den Places-Daten — Geschäftsführername und E-Mail liefert Google nicht, die trägst du bei Bedarf manuell nach.

API-Key benötigt: Google Cloud Projekt → "Places API (New)" aktivieren → Abrechnung einrichten (es gibt ein kostenloses Kontingent) → API-Key erstellen. Falls Anfragen aus diesem Fenster blockiert werden, den Key vorübergehend ohne Referrer-Beschränkung testen, aber auf "Places API" beschränkt lassen.

Einstellungen

Geräte-Synchronisierung (Handy ↔ Laptop)

Wie die Synchronisierung funktioniert

Ohne Einrichtung speichert das System deine Leads nur lokal auf dem jeweiligen Gerät. Mit einer kostenlosen Firebase-Datenbank (gehört zu Google) werden deine Leads stattdessen zentral gespeichert — dann siehst du auf dem Handy und am Laptop immer denselben Stand, solange du auf beiden Geräten mit demselben Login angemeldet bist.

Einmalige Einrichtung: Firebase-Projekt anlegen (kann dasselbe Google-Cloud-Projekt "VMG Leads" sein) → Firestore-Datenbank aktivieren → Konfiguration hier einfügen → Konto mit E-Mail/Passwort anlegen. Danach auf jedem Gerät: Datei öffnen, Konfiguration einfügen (einmalig), mit demselben Login anmelden.

Angemeldet als — Leads werden automatisch synchronisiert.
